# Demo Slack Summarizer

Summarize messages and post the digest to Slack

> Created with **Agent Maker**. This is an [Eve](https://vercel.com/eve) agent —
> a directory of files you fully own. Edit it like any repo.

## Run locally

```bash
pnpm install
pnpm dev
```

## Deploy

Deployed to your own Vercel account. Model calls bill to your team automatically
via the deployment's OIDC token (Vercel AI Gateway).

## Note

Files target the documented Eve conventions. Reconcile `agent.ts` and tool
signatures with the installed `eve` package version if they differ.
